# csgo-bsg

### Install gulp and bower
```bash
# Gulp
npm install --global gulp

# Bower
npm install --global bower

# Node modules
npm install
```


### Building 
```bash
bower install
gulp
```

### Development

To run live reload of code use

```bash
gulp reload
```

### Release
```bash
gulp release
```
